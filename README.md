Project-Rock-Paper-Scissors
A Rock, Paper, Scissors game with numerous possibilities and outcomes dependant on player choice. Test your skills to see if you can keep winning.

Requirements
Python must be installed on your computer to run this project.
Please extract Zip file using a tools such as 7-Zip or WinRaR.

Option A: Opening and running via Visual Studio Code

1. Download the repository.
2. Open folder and move to downloads section to find the file Project-Rock-Paper-Scissors-main.
3. Right click and extract all.
4. Click on new folder created and open Rock, Paper, Scissors.py.
5. If VS Code opens in "Restricted Mode", click the "Trust" (only if you trust the source of the project).
6. Click the Run button and enjoy the game.

Option B: Run this Using Command Prompt

1. Download this repository.
2. Once downloaded right click the file and extract all.
3. Open the Command Prompt.
4. Navigate to the folder containing Rock-Paper-Scissors.py.
5. You can do this by:

   cd Downloads
   cd Project-Rock-Paper-Scissors-main
   cd Project-Rock-Paper-Scissors-main

6. Once Located type py "Rock, Paper, Scissors.py". In case this does not work please try: python "Rock, Paper, Scissors.py".
